# SteelWeigh - Stainless Steel Food Scale E-commerce Website

A multi-page e-commerce website for premium stainless steel food scales built with HTML, CSS, and Bootstrap 5.

## Project Overview

SteelWeigh is a fictional e-commerce website specializing in high-quality stainless steel food scales. The website features a unique design with a modern aesthetic, focusing on simplicity and elegance without heavy images, videos, or animations.

## Features

- Responsive design that works on all devices
- Multiple product pages with detailed information
- Shopping cart functionality
- About page with company information
- Contact page with form
- Privacy policy, terms of service, and other legal pages
- GDPR compliance with cookie consent
- SEO optimized with proper meta tags
- Accessibility considerations

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome for icons

## Pages Included

- Home page (index.html)

- Individual product pages
- About page
- Contact page

- Shipping policy
- Refund policy

## Setup Instructions

1. Clone or download this repository
2. Open any HTML file in a web browser to view the website
3. No server-side setup required as this is a static website
